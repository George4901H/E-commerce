# FreshCart E-Commerce

A React e-commerce website with products, categories, brands, cart, wishlist, authentication, checkout, and product details pages.

## Live Demo

[View Live Demo](https://e-commerce-d9meyounz-george4901hs-projects.vercel.app/)

## Features

- Browse products, categories, and brands
- Search and filter products
- Product details page
- Add products to cart
- Add and remove products from wishlist
- User authentication
- Checkout and orders flow
- Responsive layout

## Tech Stack

- React
- React Router
- Axios
- Bootstrap
- Formik and Yup
- React Toastify

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm start
```

Build for production:

```bash
npm run build
```
